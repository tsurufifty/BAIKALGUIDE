# Работа с админкой Strapi

Адрес: `https://<домен>/admin`

## Роли
- **Super Admin (Admin)** — полный доступ. Создаётся при первом запуске.
- **Editor** — создание/редактирование/публикация контента и медиа.
- **Moderator** — кастомная роль: чтение контента + управление комментариями
  (поле `approved`). Создаётся в Settings → Administration Panel → Roles
  (кастомные роли бесплатны в Strapi CE с v4.8).

## Контент
Коллекции: Routes, Events, Locations, Articles, Artists.
Для каждой записи:
- заполните контент на **ru**, затем переключите локаль на **en** и добавьте перевод;
- задайте `slug`, `cover`, компонент `SEO` (в т.ч. FAQ для AI Search);
- для карты заполните `coordinates` (Geo Point: longitude/latitude);
- нажмите **Publish** (Draft не виден на сайте).

## Модерация комментариев
Комментарии создаются со статусом `approved = false` и не видны публично.
Откройте коллекцию **Comment**, проверьте текст, выставьте `approved = true`.

## Права API
Права ролей Public/Authenticated выставляются автоматически при старте backend —
вручную в Settings → Roles менять не требуется.
