# Инструкция восстановления

## База данных
```bash
gunzip -c backup_db_YYYY-MM-DD.sql.gz \
  | docker compose exec -T postgres psql -U "$DATABASE_USERNAME" -d "$DATABASE_NAME"
```

## Медиа
```bash
docker run --rm -v baikal-guide_uploads:/data -v "$PWD":/backup alpine \
  sh -c "cd /data && tar xzf /backup/backup_uploads_YYYY-MM-DD.tar.gz"
```

## После восстановления
```bash
docker compose restart backend frontend
```
Проверьте, что контент и изображения отображаются на сайте и в админке.
