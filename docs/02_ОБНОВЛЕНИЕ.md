# Инструкция обновления

1. Сделайте резервную копию (см. 03_БЭКАП).
2. Получите новый код:
   ```bash
   git pull
   ```
3. Пересоберите и перезапустите:
   ```bash
   docker compose up -d --build
   ```
4. Проверьте логи:
   ```bash
   docker compose logs -f --tail=100 backend frontend
   ```
5. Миграции схемы Strapi применяются автоматически при старте backend
   (Strapi управляет таблицами PostgreSQL).

## Откат
Если что-то пошло не так — восстановите из бэкапа (см. 04_ВОССТАНОВЛЕНИЕ) и
вернитесь на предыдущий коммит: `git checkout <prev>` + `docker compose up -d --build`.
