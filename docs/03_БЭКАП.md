# Инструкция резервного копирования

Бэкапить нужно две вещи: базу PostgreSQL и медиа (загрузки Strapi).

## База данных
```bash
docker compose exec -T postgres pg_dump -U "$DATABASE_USERNAME" "$DATABASE_NAME" \
  | gzip > backup_db_$(date +%F).sql.gz
```

## Медиа (том uploads)
```bash
docker run --rm -v baikal-guide_uploads:/data -v "$PWD":/backup alpine \
  tar czf /backup/backup_uploads_$(date +%F).tar.gz -C /data .
```

## Рекомендация
Настройте cron на ежедневный бэкап и храните копии вне сервера.
Также сохраните `.env` в защищённом месте.
