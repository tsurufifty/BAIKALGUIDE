# Инструкция запуска

## Требования
- Docker и Docker Compose
- Домен с DNS, указывающим на сервер (для SSL)

## Шаги
1. Скопируйте окружение и заполните секреты:
   ```bash
   cp .env.example .env
   # сгенерируйте значения: openssl rand -base64 32 (для APP_KEYS — 4 через запятую)
   # пропишите NEXT_PUBLIC_YANDEX_MAPS_API_KEY (Yandex Maps) и домен в *_URL/CORS_ORIGINS
   ```
2. (Опционально) первичный демо-контент: в `.env` выставьте `SEED_DATA=true`.
3. Шрифты для PDF: положите `Montserrat-Regular.ttf` и `Montserrat-SemiBold.ttf`
   в `frontend/public/fonts/` (иначе кириллица в PDF не отрисуется).
4. SSL-сертификаты: поместите `fullchain.pem` и `privkey.pem` в `docker/nginx/ssl/`.
5. Запуск:
   ```bash
   docker compose up -d --build
   ```
6. Откройте `https://<домен>/admin` и создайте первого администратора (Super Admin).
7. После первого запуска верните `SEED_DATA=false` и перезапустите backend:
   ```bash
   docker compose up -d backend
   ```

## Проверка
- Сайт: `https://<домен>/ru`
- API: `https://<домен>/api/routes`
- Админка: `https://<домен>/admin`
